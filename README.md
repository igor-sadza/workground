### About The Project

I'm starting to get a terrible headache every time when I think about setup a local dev-env. I decided to stop this madnes here - packing my beloved cli tools into an one elegant docker container.

#### Tools included

* neovim
  * coc.nvim
  * neerdtree
* kubectl
  * helm
* sdk's
  * jdk
  * cpp
  * npm
* docker-cli
